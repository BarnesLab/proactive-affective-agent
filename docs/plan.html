Core Question 2×2 Design Version Status Results Per-Version Details User Matrix Action Items Timeline

Cancer survivors complete ecological momentary assessments (EMA) several times per day, reporting their emotional state (positive affect, negative affect, discrete emotions). Meanwhile, their smartphones passively collect sensing data (screen usage, motion, keyboard, ambient light, music).


Can an LLM predict emotional states from passive sensing data alone? And does giving it access to diary narratives and autonomous tool-use improve predictions?

Key Observations

✓
Multimodal > Sensing-only (V3 vs V1)
Adding diary data improves prediction: MAE drops from 6.85 to 3.97, BA improves 0.534 → 0.617, F1 improves 0.292 → 0.472. Diary narratives are an important signal for emotional state prediction.
✓
CALLM baseline is competitive
MAE=3.53 slightly beats V3's 3.97. CALLM uses only diary (no sensing) with TF-IDF RAG retrieval. This suggests diary content alone captures most predictive signal; adding sensing may add noise.
✓
V2 early signal: agentic sensing outperforms structured sensing
V2 (partial, 187 entries): BA=0.645, F1=0.529 vs V1 (full, 427): BA=0.534, F1=0.292. Agentic tool-use approach shows strong improvement over fixed pipeline, even with sensing-only data.
ℹ
V2 running, V4 pending
V2 re-run in progress (4 eval users). V4 will start after V2 completes. Key question: does V4 (agentic + diary) beat V3 (structured + diary)?
⚠
Sensing-only approaches predict negative affect too high
V1: NA_MAE=9.91 (predicts mean ~11.2 vs GT ~1.7). V2 similar (NA_MAE=9.89). Without diary context, the LLM dramatically overestimates negative affect from passive sensing.
