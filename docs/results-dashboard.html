Key Findings

Agentic > Structured

Agents that autonomously query sensor data via tool-use (V4 V6) consistently outperform structured pipeline agents (V1 V3) that receive pre-formatted summaries.

V4 BA=0.666 vs V3 BA=0.607 (+9.7%)
V2 BA=0.598 vs V1 BA=0.521 (+14.8%)

LLM Agents Beat ML Baselines

The best agentic system (V6 BA=0.669) outperforms all traditional ML and text baselines, including MiniLM sentence embeddings (BA=0.629) and combined RF (BA=0.620).

Traditional sensing-only ML near chance (BA ~0.50).
LLM reasoning provides 33% relative improvement.

Sensing Augments Diary

Adding passive sensing to diary text significantly improves prediction. V6 (sensing+diary) outperforms CALLM (diary-only, CHI 2025) by +6.9% BA.

Behavioral signals (mobility, screen time, activity) carry complementary information beyond self-report.

Proactive Prediction Works

Sensing-only agents (V2 BA=0.598, V5 BA=0.601) predict well above chance without any diary entry, enabling intervention for participants who skip self-report.

Feasible JITAI deployment: no user burden required.

Hypothesis Evaluation

Supported

H1: Sensing data enhances diary-based prediction

V4 (sensing+diary, BA=0.666) and V6 (BA=0.669) both outperform CALLM (diary-only, BA=0.626). Passive behavioral signals carry complementary information beyond self-report text.
Supported

H2: LLM agents outperform traditional ML baselines

V6 BA=0.669 exceeds MiniLM (0.629), Combined RF (0.620), and all sensing-only ML baselines (~0.50). The LLM's contextual reasoning over heterogeneous sensor streams provides substantial lift.
Partially Supported

H3: Passive sensing alone enables meaningful prediction

V2 BA=0.598 and V5 BA=0.601 — significantly above chance (0.50) but below multimodal agents. Sensing-only prediction is feasible but diary text provides substantial additional signal.
Supported

H4: Agentic investigation outperforms structured pipelines

Consistent across both data conditions. Sensing-only: V2 BA=0.598 > V1 BA=0.521 (+14.8%). Multimodal: V4 BA=0.666 > V3 BA=0.607 (+9.7%). Selective, evidence-driven querying beats unconditional feature dumps.
Supported

H5: Diary text adds value beyond sensing

V4 BA=0.666 > V2 BA=0.598 (+11.4%). V6 BA=0.669 > V5 BA=0.601 (+11.3%). Diary entries provide emotional context that passive sensing cannot fully capture.

Example: Agentic Investigation Trace

How V4/V6 agents reason about a real participant. The agent autonomously decides which sensing data to query, building up evidence like a behavioral data scientist.

Participant 071 — 2024-01-15, 2:00 PM EMA
62-year-old female, breast cancer stage II, 4 years post-treatment. PHQ-8 = 8 (mild depression).
"I need to figure out this person's emotional state around 2pm today. She has mild depression and is not very extraverted. Let me start with a big-picture look."
get_daily_summary(date="2024-01-15")
GPS: stayed home all day, 0 km travel. Screen: 3.4h total (high).
Motion: 98% stationary. Keyboard: 12 words typed (very low). Music: no listening.
Belief update: She barely moved, barely typed, barely left home. Need to check if this is unusual for her.
"The home confinement and inactivity stand out. Let me check if this is unusual — maybe she works from home and this is normal."
compare_to_baseline(metric="gps_at_home_pct", lookback_days=14)
14-day baseline: typically home 52% of the time. Today: 100%.
Percentile: 97th. z-score: +2.8. Extremely unusual for this participant.
Belief update: Strong anomaly signal — much more confined than usual, not just "she works from home."
"97th percentile home confinement is a red flag. Paired with low keyboard use and no music — possible social withdrawal. Let me check similar past patterns."
find_similar_days(top_k=3)
2024-01-08: (home all day, high screen, no music) → PANAS_Neg=5, ER_desire=6
2023-12-22: (home, holiday) → PANAS_Neg=3, ER_desire=2
2024-01-02: (home, low GPS) → PANAS_Neg=5, ER_desire=5
Belief update: The 2024-01-08 match (same pattern as today) showed the highest negative affect. Concerning.
"Let me check recent receptivity — she rejected yesterday, which may signal intervention fatigue."
get_receptivity_history(n=7)
Last 7 days: 4 accepted, 2 rejected.
Yesterday 2pm slot: rejected (low ER_desire).
Monday pattern: often has lower affect on Mondays.
Belief update: Yesterday's rejection + Monday pattern + behavioral anomaly converge. Elevated NA likely, plus intervention fatigue.
Final Prediction
PANAS_Neg = 5 (elevated, 69th personal percentile)
ER_desire = 5 (moderate)
NA_State = True
Availability = Yes (but with caution — suggest lighter intervention)
Reasoning: Extreme home confinement (97th pct), very low typing, behavioral match to a past high-NA day (2024-01-08), plus yesterday's rejection → elevated negative affect. Recommend lower-intensity contact.
